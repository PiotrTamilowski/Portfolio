AI Image Generator

Naciśnij na obraz, żeby przejść do działającej wersji demonstracyjnej

Końcowy projekt realizowany w ramach zaliczenia studiów podyplomowych na kierunku Programista aplikacji internetowych. Celem tego projektu było stworzenie aplikacji przeglądarkowej umożliwiającej użytkownikowi, generowanie dowolnych obrazów dzięki możliwościom jakie daje sztuczna inteligencja. Dzięki temu projektowi utrwaliłem swoją wiedzę odnośnie zarówno elementów CSSa takich jak flexbox czy animacje jak i elementów języka JavaScript takich jak sposoby modyfikowania elementów DOM. Nauczyłem się korzystać we właściwy sposób z session/local storage, łączyć z api i pobierać dane z zewnętrznego serwera przy użyciu biblioteki Axios, a także korzystać z różnego rodzaju modeli AI, z którymi wcześniej nie miałem styczności.
